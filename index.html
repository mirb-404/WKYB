ena-chat

Modular Web Component chatbot. Plug in any theme, swap any LLM adapter.

Customer

Loads src/customers/<slug>.json — applies its theme, header, system prompt, and sampling overrides. Resets the conversation.

Theme override

Picks a built-in theme. A loaded customer profile overrides this with its own colors.

Connection

Browser-side API keys are visible to anyone who opens devtools. For production, proxy requests through your backend and point base-url at it.

Embed snippet

Drop this into any page: 

<script type="module" src="/src/ui/chat-element.js"></script>
<ena-chat
    theme="enadyne"
    model="qwen-plus"
    base-url="https://your-backend.example/v1">
</ena-chat>
